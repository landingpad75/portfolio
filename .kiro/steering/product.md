# Product Overview

**puffy.ink** is a personal portfolio website for Mohammed Amine Maazioui (aka Sky), a backend developer specializing in C++, TypeScript, and Assembly.

## Key Features
- Interactive developer portfolio with animated code displays
- Responsive single-page application with smooth scrolling navigation
- Dark theme with modern gradient design
- Preloader animation and smooth section transitions
- Contact form and project showcase
- Terminal-style code windows and floating code animations

## Target Audience
- Potential employers and clients
- Fellow developers and tech community
- Anyone interested in Sky's development work and projects
